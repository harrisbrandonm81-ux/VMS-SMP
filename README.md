# VMS SMP - Eaglercraft Multiplayer Server

This repository contains configuration files for running Eaglercraft multiplayer servers across multiple versions.

## Supported Versions
- 1.7.3
- 1.5.2
- 1.8.8
- 1.8
- 1.9.4
- 1.12.2

## Server Configuration

### Operator
Operator user: **brandonsocials** (Full access - level 4)

### Files
- `server.properties` - Main server configuration
- `ops.json` - Operator privileges configuration

## Setup Instructions

1. Download Eaglercraft server files for your desired version
2. Copy `server.properties` and `ops.json` to your server directory
3. Update configurations as needed
4. Start your server

## Server Settings
- Max Players: 20
- Online Mode: True
- Default World: world
- Operator: brandonsocials (Level 4 - Full permissions)
